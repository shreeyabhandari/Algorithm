#### Sorting
* Selection sort 
Strategy
At each step, put at the right position the smallest element.
* Bubble Sort 
Strategy
If a element is bigger than the next adjacent element then swap them. Repeat until no swapping.
* Insertion Sort
Strategy
Add an element to a sorted list
